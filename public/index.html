USERS
HTTP Methods PATH DESCRIPTION
GET /users Get all users
GET /users/:id Get A single user
POST /users/register Register a user
PATCH /users/update/:id Update User Info
DELETE /users/delete/:id Remove a user

PRODUCTS
HTTP PATH DESCRIPTION
GET /products Get all products
GET products/:id Get a single product
POST /products/addProduct Add/Insert A PRODUCT
PATCH /products/update/:id Update Product Info
DELETE /products/delete/:id Remove Product/Item
